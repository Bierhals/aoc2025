# Advent of Code 2025 — Lösungen

Dieses Repository dient dazu, die Aufgaben von Advent of Code 2025 (https://adventofcode.com/2025) zu lösen und die Lösungen sauber pro Tag zu versionieren.

Zweck
- Persönliches Übungsprojekt zur Problemlösung und zum Vergleich verschiedener Ansätze.
- Jede Tagesaufgabe wird als eigenes kleines Projekt/Verzeichnis angelegt, damit Lösungen, Tests und Eingabedaten getrennt bleiben.

Geplante Struktur
- Day1/, Day2/, ..., Day12/ — je ein Verzeichnis pro Tagesaufgabe.
  - Jedes Day-Verzeichnis enthält typischerweise:
    - ProjectName.csproj — die Projektdatei (.NET)
    - Program.cs — Einstiegspunkt mit der Lösung (häufig eine konsolenbasierte Lösung)
    - input.txt — den persönlichen Puzzle-Input
    - tests/ — optional; Tests werden nicht immer angelegt
